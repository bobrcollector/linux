Настройка SSH

ссылка на настройку SSH

Заходим в меню > устройства > сеть > проброс портов.
Имя - SSH, Порт хоста - 2222, Порт гостя - 22.
В терминале вводим команду для установки SSH: sudo apt install openssh-server.
Включаем службу SSH: sudo systemctl enable --now ssh.
Проверяем, что она включилась: sudo systemctl status ssh.
Если зелёненьким показывается Active - все супер.

Настройка виртуального принтера

Заходим в настройки > принтеры > добавить принтер. Там появится принтер и надо его добавить. Если этого не произошло, то в терминал пишем: sudo apt install cups-pdf либо sudo apt install cups. И снова пробуем добавить принтер

Добавление пользователя в группу

Создание пользователя: sudo useradd -m user1. Задать пароль юзеру: sudo passwd user1
 Проверить всех пользователей: ls -l /home. Проверить группы пользователя: groups user1.
Добавить группу: sudo groupadd developers. Добавить пользователя в группу: sudo useradd -aG admins user1.
Наделить пользователя правами админа: sudo usermos -aG sudo user1. Посмотреть все группы: cat /etc/group.
